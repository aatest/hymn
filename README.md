# ♪ 악보 라이브러리 — 찬송가 · 복음성가 악보 뷰어

교회 예배 및 찬양팀을 위한 악보 이미지 뷰어입니다.
서버 없이 브라우저에서 바로 실행되며, 악보 이미지를 등록하고 관리할 수 있습니다.

## 시작하기

`hymn-viewer_20260313_2.html` 파일을 브라우저(Chrome, Edge, Safari 등)에서 열면 바로 사용할 수 있습니다.

> 별도 설치나 서버 구동이 필요 없습니다.

## 주요 기능

### 악보 관리
- **카탈로그에서 추가** — 새찬송가 645곡, 복음성가 487곡 목차에서 검색 후 클릭으로 추가
- **직접 추가** — 곡 제목, 번호, 악보 이미지를 수동 등록
- **폴더 일괄 등록** — 악보 이미지 폴더를 선택하면 파일명에서 번호를 자동 인식하여 매핑
- **드래그 정렬** — 목록에서 곡 순서를 자유롭게 변경
- **검색 & 필터** — 번호 또는 제목으로 검색, 찬송가/복음성가 필터링

### 악보 뷰어
- 악보 이미지 확인 및 페이지 넘기기
- 줌 인/아웃 (30% ~ 300%)
- 곡 간 연속 이동 (이전곡/다음곡)
- 하단 썸네일 스트립

### 저장 & 공유
- 곡 목록을 JSON 파일로 저장 (이미지 포함)
- JSON 파일을 불러와 기존 목록에 추가하거나 교체

## 조작법

### 키보드

| 키 | 동작 |
|----|------|
| `←` `→` | 이전/다음 페이지 |
| `Shift` + `←` `→` | 이전/다음 곡 |
| `+` / `-` | 확대 / 축소 |
| `0` | 원래 크기로 |

### 터치 (모바일/태블릿)
- 좌우 스와이프로 페이지 이동

## 악보 이미지 폴더 사용법

사이드바의 **🗂 악보 폴더** 영역에서 악보 이미지가 담긴 폴더를 선택하면, 파일 이름에서 찬송가 번호를 자동 인식합니다.

### 지원하는 파일명 형식

```
093.jpg          → 93장
93.png           → 93장
찬송가093.jpg     → 93장
새찬송가093.jpg   → 93장
hymn093.jpg      → 93장
093-1.jpg        → 93장 1페이지
093-2.jpg        → 93장 2페이지
```

- 파일명에서 첫 번째 숫자 덩어리를 번호로 인식합니다 (1~800 범위)
- `-` 또는 `_` 뒤의 숫자는 페이지 번호로 인식합니다
- 같은 번호의 여러 파일은 페이지 순서대로 자동 정렬됩니다

## 데이터 저장

- 등록한 곡과 이미지는 **브라우저의 IndexedDB**에 자동 저장됩니다
- 브라우저를 닫아도 데이터가 유지됩니다
- 다른 기기로 옮기려면 **💾 저장** 버튼으로 JSON 파일을 내보낸 뒤, 다른 기기에서 **📂 불러오기**로 가져오세요
- 브라우저 데이터를 삭제하면 등록한 곡도 삭제되므로, 중요한 목록은 JSON으로 백업해 두세요
