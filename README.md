# ♪ 찬송가 · 복음성가 악보 뷰어

단일 HTML 파일로 동작하는 악보 관리 앱입니다.  
설치 없이 브라우저에서 바로 사용할 수 있으며, 아이패드 사파리에서도 동작합니다.

---

## 시작하기

### 로컬 사용
`hymn-viewer.html` 파일을 브라우저로 열면 바로 사용 가능합니다.

### 아이패드(사파리) 사용
GitHub Pages를 통해 접속합니다.

1. GitHub 저장소에 `hymn-viewer.html` 업로드
2. Settings → Pages → Branch: main 저장
3. 생성된 주소를 사파리에서 접속

```
https://아이디.github.io/저장소명/hymn-viewer.html
```

사파리 주소창에서 **공유 → 홈 화면에 추가** 하면 앱처럼 사용할 수 있습니다.

---

## 주요 기능

### 악보 목록 관리
- **악보 추가**: 구분(찬송가/복음성가), 번호, 제목, 이미지 등록
- **찬송가 목차에서 악보 추가**: 새찬송가 1~645장 전체 검색 후 원클릭 등록
- **복음성가 목차에서 악보 추가**: 복음성가·CCM 198곡 검색 후 원클릭 등록
- **검색**: 번호 또는 제목 키워드로 실시간 검색
- **필터**: 전체 / 찬송가 / 복음성가 탭 전환
- **순서 변경**: 드래그 앤 드롭 (PC)
- **삭제**: 항목 위에 마우스 올리면 ✕ 버튼 표시

### 악보 이미지 폴더 자동 연결 *(신규)*
- 사이드바 상단 **🗂 악보 폴더** 에서 이미지가 들어있는 폴더를 선택
- 폴더 안의 이미지 파일을 자동으로 읽어 찬송가 번호에 매핑
- 목차에서 악보 추가 시 해당 번호의 이미지가 **자동으로 함께 등록**
- 목록에 이미지 보유 여부(`🖼 2p`) 및 미보유(`이미지 없음`) 표시

**지원하는 파일명 형식:**

| 파일명 예시 | 인식 결과 |
|------------|----------|
| `093.jpg`, `93.jpg` | 93장 |
| `찬송가093.jpg`, `새찬송가_093.jpg` | 93장 |
| `hymn093.jpg` | 93장 |
| `093-1.jpg`, `093-2.jpg` | 93장 1·2페이지 (순서 자동 정렬) |

### 악보 뷰어
- 이미지 한 화면에 맞게 자동 표시
- 이전/다음 페이지 이동 (버튼 또는 하단 썸네일 클릭)
- 확대/축소 (+/- 버튼 또는 키보드)
- 하단 썸네일 **＋** 버튼으로 페이지 추가

### 곡 간 연속 이동
- 뷰어 상단 **«** / **»** 버튼으로 이전/다음 곡 이동
- 마지막 페이지에서 **›** 누르면 다음 곡 첫 페이지로 자동 이동
- 첫 페이지에서 **‹** 누르면 이전 곡 마지막 페이지로 자동 이동

### 저장 / 불러오기
- **💾 저장**: 이미지 포함 전체 목록을 `.json` 파일로 내보내기
- **📂 불러오기**: 저장한 `.json` 파일을 불러와 목록 복원
  - 기존 목록이 있을 경우 **추가** 또는 **교체** 선택 가능
  - 중복 항목은 자동 제외

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `←` / `→` | 이전/다음 페이지 (곡 경계 자동 이동) |
| `Shift + ←` / `Shift + →` | 이전/다음 곡 |
| `+` | 확대 |
| `-` | 축소 |
| `0` | 원래 크기 |

---

## 데이터 저장 방식

- 목록 데이터는 브라우저 **IndexedDB**에 자동 저장됩니다. (용량 제한 없음)
- 이미지는 **Base64**로 변환되어 저장됩니다.
- 기존에 **localStorage**에 저장된 데이터는 첫 실행 시 자동으로 IndexedDB로 마이그레이션됩니다.
- 저장/불러오기 기능으로 기기 간 데이터 이동이 가능합니다.
- GitHub Pages 사용 시 데이터는 해당 기기 브라우저에 저장됩니다.
